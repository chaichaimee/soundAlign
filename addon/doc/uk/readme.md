## Огляд
**Переосмисліть свій світ звуків із SoundAlign!** Найкращий додаток для NVDA, що надає повний контроль над **стереопанорамуванням, формами хвилі та алгоритмами загасання.**

<br>

## Гарячі клавіші
> **NVDA + Windows + S** – Одне натискання: відкрити налаштування.  
> **NVDA + Windows + S** – Подвійне натискання: увімкнути/вимкнути SoundAlign.

<br>

## Особливості
* **Точне панорамування:** Лівий, центральний або правий канали для сповіщень.
* **12 алгоритмів загасання:** Косинус, Лінійний, S-крива та багато інших.
* **Майстер-гучність:** Глобальний регулятор (0–100%) для всіх звуків додатка.
* **Інтелектуальні оголошення:** Голосові та звукові сповіщення про прогрес.
