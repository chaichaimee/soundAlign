## Resumen
**¡Redefine tu mundo auditivo con SoundAlign!** SoundAlign es el complemento definitivo para NVDA diseñado para transformar cómo percibes las respuestas sonoras mediante **paneo estéreo, formas de onda avanzadas y algoritmos de desvanecimiento profesionales.**

<br>

## Teclas Rápidas
> **NVDA + Windows + S** – Un toque: abrir configuración.  
> **NVDA + Windows + S** – Dos toques: activar/desactivar SoundAlign.

<br>

## Características
* **Paneo de Sonido de Precisión:** Dirige sonidos a los canales Izquierdo, Centro o Derecho.
* **12 Algoritmos de Desvanecimiento:** Coseno, Lineal, Exponencial, Logarítmico, etc.
* **Control de Volumen Maestro:** Ajuste global (0–100%) para sonidos de SoundAlign.
* **Anuncios Inteligentes:** Configura intervalos de voz y pitidos para el progreso.
