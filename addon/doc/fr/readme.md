## Aperçu
**Redéfinissez votre univers auditif avec SoundAlign !** L'extension NVDA ultime pour maîtriser le **panoramique stéréo, les formes d'onde et les fondus sonores.** Sculptez chaque son de votre lecteur d'écran pour une navigation immersive.

<br>

## Touches de raccourci
> **NVDA + Windows + S** – Un appui : ouvrir les paramètres.  
> **NVDA + Windows + S** – Double appui : activer/désactiver SoundAlign.

<br>

## Fonctionnalités
* **Panoramique de précision :** Canaux Gauche, Centre ou Droite pour les alertes et effets.
* **12 Algorithmes de fondu :** Cosinus, Gaussien, Linéaire, Exponentiel, etc.
* **Volume Général :** Curseur global (0–100%) pour tous les sons traités.
* **Annonces de progression :** Intervalles de parole et bips personnalisables.
