## Übersicht
**Definieren Sie Ihre Audiowelt mit SoundAlign neu!** SoundAlign ist das ultimative NVDA-Add-on, das Ihre Wahrnehmung von Feedback transformiert. Durch die volle Kontrolle über **Stereo-Panning, fortschrittliche Wellenformen und 12 Fade-Algorithmen** können Sie nun jeden Ton Ihres Screenreaders räumlich und klanglich gestalten.

<br>

## Tastenkombinationen
> **NVDA + Windows + S** – Einmal drücken: Einstellungen öffnen.  
> **NVDA + Windows + S** – Zweimal drücken: SoundAlign ein-/ausschalten.

<br>

## Funktionen
* **Präzises Sound-Panning:** Töne links, mittig oder rechts positionieren (Fehler, Effekte, Navigation).
* **12 Profi-Fade-Algorithmen:** Cosinus, Linear, Exponentiell, S-Kurve und mehr.
* **Master-Lautstärkeregelung:** Globaler Regler (0–100%) für alle SoundAlign-Töne.
* **Wellenform-Auswahl:** Sinus, Dreieck, Sägezahn, Rechteck oder Original-Beep.
