## Обзор
**Переосмыслите свой мир звуков с SoundAlign!** Ультимативное дополнение для NVDA, дающее полный контроль над **стереопанорамированием, формами волны и алгоритмами затухания.**

<br>

## Горячие клавиши
> **NVDA + Windows + S** – Одно нажатие: открыть настройки.  
> **NVDA + Windows + S** – Двойное нажатие: вкл/выкл SoundAlign.

<br>

## Функции
* **Точное панорамирование:** Левый, центральный или правый каналы для ошибок и эффектов.
* **12 алгоритмов затухания:** Косинус, Линейный, Экспоненциальный и другие.
* **Общая громкость:** Мастер-регулятор (0–100%) для всех звуков дополнения.
* **Умные оповещения:** Настройка голосовых и звуковых интервалов прогресса.
